This is the overarching structure of the application. Feel free to add features on top of this, but at its core, make sure all of this functionality exists.


📂 CODE: Club OS Blueprint (Finalized Plan)
Organization: CODE (~30 Members + Alumni)

Primary Partner: Seven Hills Foundation

🏛️ 1. Governance & Membership (The Admin View)
Admin Command Center:

Inquiry Inbox: Manage member questions with a Category Dropdown (Technical, Internship, Events). E-Board can reply privately (sent via WhatsApp DM) or publish to the Anonymous FAQ Page.

Attendance Red Flags: Configurable by role (e.g., Leads vs. Members). Toggles for automated WhatsApp check-ins or manual E-Board reach-out.

Approval Queue: E-Board gatekeeps all Project Pitches and Resource Library uploads.

Profiles: Feature a "Standout Skill" (user-selected), full project role history, and Engagement Metrics (Attendance, Pitches, Uploads).

Alumni: Read-only access to resources; "Mentorship" toggle reveals LinkedIn/Email.

🛠️ 2. The Project Ecosystem (CODE for a Cause)
Dual-View Dashboards:

External (Client): Seven Hills sees a high-level status: In Progress or Complete.

Internal (Team): Full milestone tracking (Ideation, Design, Dev, Review) and an Internal Notes section for everyone on the team to log blockers/bugs.

AI-Sourced Case Studies: Team Lead submits individual member reflections (based on 4 prompts: Hardest Bug, Learning, Fun Part, Additional Thoughts). AI synthesizes these into a professional draft for E-Board review.

The Hall of Fame: Public-facing portfolio featuring the Featured Project, Live Demos, Videos, and Git links.

📚 3. The Resource Library & FAQ
The Academy: Tiered into "Official CODE Guides" and "Community Resources" with a global Search Bar and Phase/Topic toggles.

FAQ Page: A dedicated, searchable space for anonymous, E-Board-vetted club questions.

Live Feed (Active Only): Clickable real-time stream of milestone shifts and uploads. E-Board can Pin posts to the top (no automatic WhatsApp trigger for pins).

📢 4. Communication & Automation (The "CODE-Bot")
WhatsApp Routing: Specific Group IDs provided by E-Board allow the bot to route Seven Hills feedback directly to Project Subgroups.

Broadcasting: Monday Morning Digest and 24-hour Deadline alerts sent to the Main Group.

Inquiry Loop: Delivers private E-Board responses and "Red Flag" check-ins via DM.
